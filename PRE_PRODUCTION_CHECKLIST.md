# Pre-Production Checklist

## Seguridad
- [ ] Variables de entorno configuradas en Vercel
- [ ] Claves API restringidas a dominios específicos
- [ ] RLS policies activas en Supabase
- [ ] SSL/TLS habilitado en dominio

## SEO
- [ ] Meta tags completados
- [ ] Sitemap.xml generado y accesible
- [ ] Robots.txt configurado
- [ ] Open Graph images generadas
- [ ] Structured data (JSON-LD) implementada
- [ ] Canonical URLs definidas
- [ ] Hreflang tags para internacionalización

## Performance
- [ ] Imágenes optimizadas (WebP/AVIF)
- [ ] Lazy loading implementado
- [ ] Code splitting configurado
- [ ] Service Worker registrado
- [ ] Cache strategy definida

## PWA
- [ ] Manifest.json configurado
- [ ] Iconos en todos los tamaños
- [ ] Offline page funcional
- [ ] Push notifications configuradas

## Funcionalidad
- [ ] Flujo de registro/login probado
- [ ] Publicación de productos probada
- [ ] Sistema de chat probado
- [ ] Pagos/créditos probados
- [ ] Moderación automática probada

## Legal
- [ ] Términos y condiciones actualizados
- [ ] Política de privacidad actualizada
- [ ] GDPR compliance (si aplica)
- [ ] Aviso de cookies implementado

## Monitoreo
- [ ] Vercel Analytics habilitado
- [ ] Speed Insights configurado
- [ ] Error tracking (Sentry) configurado
- [ ] Uptime monitoring activado

## Dominio
- [ ] DNS configurado correctamente
- [ ] SSL certificate activo
- [ ] Redirects configurados
- [ ] WWW/non-WWW consistency

## Base de Datos
- [ ] Backup automático configurado
- [ ] Índices optimizados
- [ ] Queries lentas revisadas
- [ ] Connection pooling activado
